## OFFICIAL vs COMMUNITY SERVERS

**All science/research MCP servers found are COMMUNITY-BUILT.** No major scientific database vendor (NCBI, EBI, RCSB, ChEMBL, PubChem, UniProt, etc.) has released an official MCP server as of March 2026.

Notable official MCP servers in adjacent areas:
- EMBL-EBI is collaborating on BioContextAI but hasn't released official individual database MCP servers
- Benchling (major ELN) has not released an MCP server
- LabArchives, eLabFTW — no MCP servers found

- **Total MCP servers cataloged:** ~60+ for scientific research and lab applications
- **Most popular:** arxiv-mcp-server (2,400★), zotero-mcp (2,100★), jupyter-mcp-server (969★), paper-search-mcp (917★)
- **Prolific builder:** Augmented Nature (~15 science servers)
- **Key platforms:** bio-mcp org (~10 bioinformatics servers), MCPMed hub, Smithery.ai marketplace
- **All community-built** — no official vendor MCP servers for scientific databases
- **Strongest coverage:** Academic literature search, protein/structural biology, chemistry/drug discovery
- **Weakest coverage:** Lab equipment/instruments, LIMS/ELN, scientific workflows, research data repositories, materials science

---

## KEY TRENDS

1. **Bioinformatics leads adoption** — Most mature MCP ecosystem, with comprehensive protein/genomics/chemistry coverage
2. **Multi-source aggregation** — Trend toward servers that combine 5-25+ sources (paper-search-mcp, Academix, Scientific-Papers-MCP)
3. **Drug discovery pipeline** — Near-complete MCP coverage: PubChem → ChEMBL → AlphaFold → ClinicalTrials.gov → OpenFDA
4. **EMBL-EBI engagement** — Institutional interest signals future official adoption
5. **Lab bench gap** — Physical laboratory work (instruments, LIMS, ELN) remains almost entirely disconnected from MCP
6. **Security concerns** — Tool poisoning and cross-server shadowing identified as risks
7. **Standardization emerging** — MCPMed academic paper, BioinfoMCP auto-generation, bio-mcp templates all pushing toward standardized approaches
8. **R statistics integration** — rmcp with 52 tools and 429 packages represents deep statistical computing access
