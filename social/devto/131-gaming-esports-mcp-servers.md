**At a glance:** 25+ gaming and esports MCP servers across 7 subcategories. All community-built — no official platform MCPs. **Rating: 3.5/5.**

**Headlines:** Minecraft has the highest-starred gaming MCP (504 stars) with real-time character control. OP.GG provides the richest esports data (27 tools across 3 Riot titles). Chess has the most implementations for a single game (4 servers).

## Live Game Interaction

- **yuniko-software/minecraft-mcp-server** (504 stars) — AI-controlled Minecraft bot via Mineflayer API. Movement, building, mining, inventory, furnace ops, entity detection, in-game chat.
- **notpoiu/roblox-mcp** (18 stars, 18 tools) — interacts with running Roblox game clients. Code execution, script decompilation, remote interception, multi-client support.

## Esports Analytics

- **opgginc/opgg-mcp** (76 stars, 27 tools) — **League of Legends** (profiles, champions, counters, meta), **TFT** (decks, items, builds), **Valorant** (agents, maps, leaderboards), **esports schedules**. No API key required.

## Steam

- **dsp/mcp-server-steam** (12 stars) — Steam API gaming context (Java)
- **algorhythmic/steam-mcp** (4 stars, 10 tools) — player achievements, game stats, news, store details
- **fenxer/steam-review-mcp** (5 stars) — review analysis with sentiment

Three separate servers, none offering a unified Steam experience.

## Chess (4 Implementations)

- **pab1it0/chess-mcp** (64 stars, 10 tools) — Chess.com data, player profiles, PGN downloads
- **arvid-berndtsson/Chess-MCP** (12 tools) — play against AI (levels 1-5), FEN/PGN support
- **turlockmike/chess-mcp** — Stockfish-powered analysis
- **danilop/chess-support-mcp** — game state management, intentionally no move suggestions

## Game Server Management

- **v9rt3x/cs2-rcon-mcp** (9 stars, 5 tools) — natural language CS2 server control via RCON. Workshop maps, server monitoring.

## Game Databases & Streaming

- **bielacki/igdb-mcp-server** (3 stars, 4 tools) — IGDB/Twitch game metadata, trending titles
- **moonolgerd/game-mcp** — discovers installed games across Steam/Epic/GOG/Xbox/EA/Ubisoft
- **mtane0412/twitch-mcp-server** (13 tools) — Twitch Helix API for channels, streams, clips, emotes

## What's Missing

- No official MCPs from Steam/Valve, Epic, PlayStation, Xbox, or Nintendo
- No cross-platform achievement tracking
- Limited esports coverage (only Riot Games titles)
- No in-game economy/marketplace tools
- No cloud gaming (Xbox Cloud, GeForce NOW)
- No modding workflows (Nexus Mods, Steam Workshop)

## The Bottom Line

**Rating: 3.5/5** — Creative community implementations with genuine utility. Minecraft bot is a compelling MCP demo, OP.GG shows how gaming APIs should be wrapped, Chess has surprisingly deep coverage. Biggest limitation: every server is community-built on public APIs with no official platform support.
