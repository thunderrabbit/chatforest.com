**At a glance:** 30+ education MCP servers across LMS integration, flashcards, academic research, enterprise learning, assessment, and STEM tools. Canvas LMS has the most competition (6+ servers). arXiv MCP is the highest-starred at 2,400. Udemy is the first major MOOC to ship official MCP. **Rating: 3.5/5.**

## LMS Integration

**DMontgomery40/mcp-canvas-lms** (~89 stars, TypeScript) — 54 tools covering students (assignments, grades, discussions), instructors (course creation, grading, quizzes), and admins (user management, reporting). Streamable HTTP transport, Docker deployment.

**vishalsachdev/canvas-mcp** (~75 stars, Python/TypeScript, MIT) — 90+ tools with **FERPA-compliant data anonymization**. 8 agent skills, hosted option. Learning designer capabilities with accessibility auditing. The most feature-rich Canvas implementation.

**peancor/moodle-mcp-server** (~31 stars, JavaScript, MIT) — 11 tools: courses, student management, assignments, quiz feedback. Genuinely useful for instructors managing large courses.

**faizan45640/google-classroom-mcp-server** (~2 stars, MIT) — 3 read-only tools. Minimal for a platform with 150M+ users.

**bencered/d2l-mcp-server** — 12 tools for D2L Brightspace with automated SSO.

## Flashcards & Spaced Repetition

**ankimcp/anki-mcp-server** (~179 stars, TypeScript, AGPL-3.0) — 20+ tools across review sessions, deck/note/media/template management. Local, HTTP, or ngrok-tunneled deployment. Read-only safety mode.

**arielbk/anki-mcp** (~9 stars, TypeScript, MIT) — 7 tools: quizzing, PDF-to-flashcards, analytics, bulk operations.

## Academic Research

**blazickjp/arxiv-mcp-server** (~2,400 stars, Python, Apache-2.0) — 4 tools: paper search, download, list, read. Research workflow prompts. The highest-starred education MCP by far.

**openags/paper-search-mcp** (~791 stars, Python, MIT) — **7 academic platforms in one server**: arXiv, PubMed, bioRxiv, medRxiv, Google Scholar, IACR, Semantic Scholar. Standardized output format.

## Enterprise & Assessment

**Udemy Business MCP** (official) — First major MOOC with MCP. On-demand learning in AI workflows. Pre-built connectors for ChatGPT, Claude, Cursor.

**EduBase/MCP** (~24 stars, TypeScript, MIT) — Dynamic API tools, LaTeX quiz support, SCORM modules, SSO, GDPR compliance. Institutional-grade assessment.

## STEM Tools

**Wolfram Alpha MCP** — Multiple implementations (TypeScript, Python, Go) for math, science, engineering computation. Delegates numerical work LLMs struggle with.

## What's Missing

- **Blackboard, Sakai, Open edX** — zero MCP implementations
- **Coursera, Khan Academy, LinkedIn Learning** — no integrations
- **Student information systems** — no Banner, PowerSchool connectivity
- **Academic integrity** — no Turnitin or plagiarism detection
- **K-12** — Google Classroom support is minimal (3 read-only tools)

## Bottom Line

**Rating: 3.5/5** — Canvas and Moodle coverage is decent, academic research tools are strong (arXiv at 2,400 stars), Anki integration is mature, and Udemy's official server adds enterprise credibility. But Blackboard's absence is conspicuous, K-12 is underserved, and missing SIS/library/adaptive learning tools prevent institutions from building complete AI workflows.
