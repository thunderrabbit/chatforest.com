**At a glance:** 35+ calendar and scheduling MCP servers across 7 subcategories. Google Calendar leads implementations, Microsoft ships an official server, Apple Calendar via apple-mcp (3,000 stars). **Rating: 4/5.**

## Google Calendar

### google-calendar-mcp — 1,000 stars
The standout. **Multi-account** with cross-account conflict detection. 12 tools covering the full calendar lifecycle. Advanced features: recurring event modification, natural language scheduling, intelligent event import from images/PDFs/web links. Screenshot a conference flyer → calendar events. v2.6.1 (March 2026), actively maintained.

### Other Google Calendar servers
- **calendar-mcp** (25 stars) — mutual meeting slot detection, daily busyness analysis
- **mcp-google-calendar** (9 stars) — auto timezone detection, conflict checking
- 6+ total implementations reflecting Google Calendar's API accessibility

## Microsoft Outlook / 365

### Microsoft 365 CalendarTools (Official)
**Official remote MCP server** from Microsoft — part of their 2,800-star MCP mono-repo. Hosted at Microsoft's infrastructure with tenant-scoped auth. Create, update, delete events, manage invites, check availability via Graph API. One of the first major productivity vendors with an official MCP server.

### Community Outlook servers
- **Outlook_Calendar_MCP** (33 stars) — Windows-only local access, 6 tools
- **office-365-mcp-server** (12 stars) — 24 tools across email/calendar/Teams/Planner

## Apple Calendar

**apple-mcp** (3,000 stars) — the most-starred server in the calendar space, though calendar is one of seven app integrations. Also: mcp-ical (278 stars, natural language macOS Calendar), apple-calendar-mcp (EventKit native).

## Universal Access

**CalDAV MCP** (56 stars) — works with any standards-compliant calendar: Nextcloud, Radicale, Baikal, iCloud. 4 tools. This is the "works everywhere" option.

## Scheduling Platforms

- **Cal.com** — official server, 18 stars, 9 booking management tools
- **Calendly** — universal MCP for event/invitee management
- **scheduler-mcp** (54 stars) — cron-based task automation, 12 tools

## The Killer Feature

**Mutual free slot detection** — AI finds meeting times across multiple calendars without human back-and-forth. Available in google-calendar-mcp, calendar-mcp, and office-365-mcp-server.

## What's Missing

No official Google or Apple MCP servers. AI-native scheduling platforms (Clockwise, Reclaim, Motion) have minimal MCP presence.

**Rating: 4/5** — Broad coverage across all major calendar platforms with an official Microsoft server, strong community implementations for Google Calendar, and CalDAV universal access. Loses a point for missing official Google/Apple servers.
