**At a glance:** Every major ML platform has an MCP server, but they vary enormously in scope. HuggingFace is the most polished. Ollama has the richest local inference story. No single server covers the full ML lifecycle.

## Model Hubs & Inference

**[HuggingFace hf-mcp-server](https://github.com/huggingface/hf-mcp-server)** (206 stars, official) — Hub search, documentation search, and Gradio Space execution. The flagship ML MCP server.

**[Ollama MCP](https://github.com/rawveg/ollama-mcp)** (144 stars) — 14 tools covering the complete Ollama SDK. Best local inference MCP server.

**[Replicate MCP](https://github.com/deepfates/mcp-replicate)** (93 stars) — 13 tools for cloud model marketplace. Official hosted server now available with Code Mode.

**[OpenAI bridge](https://github.com/pierrebrunelle/mcp-server-openai)** (79 stars) — Cross-model queries from Claude to GPT-4.

## Experiment Tracking

**[W&B MCP](https://github.com/wandb/wandb-mcp-server)** (41 stars, official) — 6 tools: experiment queries, trace analysis, report generation.

**MLflow MCP** — 17 tools for experiment search, run comparison, model registry, deployment.

## Training & Cloud

**PyTorch Lightning MCP** — Training, hyperparameters, checkpoints. **AWS SageMaker MCP** (official) — Endpoint and job management. **LM Studio** — MCP host + server bidirectional support.

**Rating: 3.5/5** — Strong platform coverage, fragmented lifecycle.
