**At a glance:** Google's official endpoint + community servers (1,700 stars lead). **Rating: 3.5/5.**

## Key Options

- **Google's Official Workspace MCP** — first-party, no third-party token handling
- **taylorwilsdon/google_workspace_mcp** (1,700 stars) — 12 services, three-tier permissions (Core/Extended/Full)
- **mcp-gsuite** (477 stars) — Gmail + Calendar only, narrower OAuth scope
- **mcp-headless-gmail** (53 stars) — server/container deployments without browser

## Security Considerations

OAuth scope creep, local token storage, send permission risks, no official MCP reference server. Start read-only; enable send after building trust.

## The Bottom Line

**Rating: 3.5/5** — Mature options available. Email sensitivity keeps ceiling lower than less sensitive categories.
