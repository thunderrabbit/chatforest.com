**At a glance:** [google/mcp](https://github.com/google/mcp) (3.4k stars) + [google_workspace_mcp](https://github.com/taylorwilsdon/google_workspace_mcp) (1.9k stars). **Rating: 4/5.**

## Official Google MCP

**[google/mcp](https://github.com/google/mcp)** — 3.4k stars, Apache-2.0. Managed remote endpoints for BigQuery, Cloud SQL, GKE, Maps, and more. Open-source servers for Workspace (Docs, Sheets, Slides, Calendar, Gmail), Firebase, Cloud Run.

## Community Implementations

- **[google_workspace_mcp](https://github.com/taylorwilsdon/google_workspace_mcp)** (1.9k stars, Python) — 12 Google services, OAuth 2.0/2.1, multi-user, Docker/K8s
- **[google-docs-mcp](https://github.com/a-bonus/google-docs-mcp)** (403 stars, TypeScript) — Deep Docs editing, Markdown round-trip, 173 commits
- **[mcp-gdrive](https://github.com/isaacphi/mcp-gdrive)** (272 stars) — Sheets specialist
- **[google-drive-mcp](https://github.com/piotr-agier/google-drive-mcp)** (81 stars) — Drive + Calendar with Shared Drives

## Key Issues

- OAuth setup complexity (Google Cloud project + APIs + consent screen)
- 10+ implementations with overlapping features
- Archived Anthropic reference causes confusion

## Bottom Line

Strongest cloud storage MCP ecosystem. Official Google commitment (3.4k stars) plus production-ready community server covering 12 Workspace services. Free accounts work with all community servers. **Rating: 4/5.**
