**At a glance:** The annotation MCP ecosystem is thin. Only Label Studio has a dedicated server. Most major platforms haven't built MCP servers yet.

## Label Studio

**[HumanSignal/label-studio-mcp-server](https://github.com/HumanSignal/label-studio-mcp-server)** (28 stars, Python, Apache-2.0) — Official server covering the full labeling workflow: project management, task import, prediction integration (pre-labeling cuts annotation time 50-80%).

## Labelbox

MCP as a **client-side protocol** for evaluating AI agent tool-calling. Annotators review and correct tool interactions. Not a general annotation server.

## Roboflow

**[Roboflow MCP](https://mcp.pipedream.com/app/roboflow)** — CV annotation via Pipedream. Dataset management, training, deployment.

## What's Missing

CVAT, Supervisely, Encord, V7 Darwin, Scale AI — none have MCP servers. The biggest gap in the ecosystem.

**Rating: 2.5/5** — Label Studio carries the category alone.
