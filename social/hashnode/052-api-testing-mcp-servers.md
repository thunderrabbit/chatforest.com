**At a glance:** Strong entries at every layer. [Postman](https://github.com/postmanlabs/postman-mcp-server) (100+ tools, hosted), [Apollo](https://github.com/apollographql/apollo-mcp-server) (272 stars, Rust), [mcp-graphql](https://github.com/blurrah/mcp-graphql) (365 stars), [protoc-gen-go-mcp](https://github.com/redpanda-data/protoc-gen-go-mcp) (187 stars, gRPC).

## By Protocol

- **REST**: Postman (100+ tools, 3 modes), dkmaker (single tool, quick testing), cocaxcode (20 tools, load testing, zero cloud)
- **GraphQL**: Apollo (operation-to-tool, semantic search, Rust), mcp-graphql (generic, mutations off by default)
- **OpenAPI**: awslabs (highest quality, dynamic generation), Swagger-MCP (code generation approach)
- **gRPC**: protoc-gen-go-mcp (compiler plugin, zero boilerplate), ggRMCP (runtime reflection)

## Missing

No Bruno or Hoppscotch servers, no WebSocket/SSE testing, no contract testing, no serious load testing.

**Rating: 4.0/5** — Breadth and quality of top entries, offset by gaps in contract testing and WebSocket support.
