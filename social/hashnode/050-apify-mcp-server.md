**At a glance:** [Apify MCP Server](https://github.com/apify/apify-mcp-server) gives AI agents access to 3,000+ pre-built scrapers and automation tools. 945 stars, TypeScript, Apache 2.0. Dynamic tool discovery — agent finds and uses new scrapers at runtime.

## How It Works

17+ tools: search the Apify Store, inspect Actor details, run scrapers, access stored results. The standout is **dynamic discovery** — capabilities grow at runtime based on what the agent needs.

Hosted mode at `mcp.apify.com` (OAuth, output schema inference) or local via `npx @apify/actors-mcp-server`.

## Trade-offs

- **Platform dependency** — gateway to a commercial SaaS, not standalone
- **Actor quality varies** — community-created, from excellent to abandoned
- **Costs surprise** — autonomous agents burn credits fast
- **18.8% success rate** in 250-agent stress test (vs Bright Data's 76.8%)
- Free tier: $5/mo credits, no card required

**Rating: 3.5/5** — Unique marketplace model with dynamic discovery. Free tier worth trying; watch credit consumption.
