**At a glance:** 55+ food and restaurant MCP servers. Exceptional official vendor adoption. **Rating: 4.0/5.**

## Recipes

- **HowToCook-mcp** (569 stars) — smart meal planning with dietary restrictions, built on 70K-star cooking guide
- **Spoonacular MCP** (official) — 6 tools, built by API creator
- **CookWith.co** — recipe generation and transformation, no API key

## Nutrition

- **OpenNutrition** (122 stars) — 300K+ foods, fully local, no API calls
- **Edamam** (official) — food image analysis, barcode lookup

## Restaurants

- **Yelp MCP** (official) — agent-to-agent design with natural language + multi-turn
- **restaurant-mcp** — unified Resy + OpenTable with reservation sniper

## Food Delivery

- **Swiggy** (official) — 3 services including first quick-commerce MCP (40K+ SKUs)
- **Zomato** (official) — ordering, tracking, QR payment
- **Uber Eats** (217 stars) — browser automation proof-of-concept
- India leads; Western platforms only have community implementations

## Grocery

- **Instacart** (official) — recipe → shopping list → order workflow
- **Kroger MCP** — shopping path optimization through stores

## The Bottom Line

**Rating: 4.0/5** — Most commercially embraced MCP category. Full food lifecycle served: recipe → shopping → delivery. Weaknesses: duplicated nutrition servers, absent US grocery chains, Western delivery platforms lagging.
