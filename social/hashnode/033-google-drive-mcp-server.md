**At a glance:** google/mcp (3.4k stars, official) + google_workspace_mcp (1.9k stars, community). Google Drive is the most-used cloud storage service with 1B+ users.

Google Drive MCP servers let AI agents search, read, create, and edit documents across your Google Workspace — Docs, Sheets, Slides, Calendar, Gmail — through natural language.

## Top Implementations

- **google/mcp (3.4k stars)** — Official managed remote endpoints for Cloud and Workspace. Apache-2.0.
- **google_workspace_mcp (1.9k stars)** — 12 Google services, multi-user OAuth 2.1, Docker, Kubernetes. Most comprehensive community server.
- **google-docs-mcp (403 stars)** — Deep document editing: surgical text edits, table management, Markdown round-trip. 173 commits.

## What's Good

Official Google commitment. 1.9k-star community server covering 12 services in production-ready package. Deep document editing. Free accounts work everywhere.

## What's Not

OAuth setup complexity. Archived Anthropic reference server causes confusion. 10+ overlapping implementations. Format conversion imperfect for complex docs.

## Rating: 4/5

Strongest MCP ecosystem for any cloud storage service. Loses a point for OAuth complexity and ecosystem fragmentation.
