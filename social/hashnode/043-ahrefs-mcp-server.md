**At a glance:** Official Ahrefs remote MCP server. OAuth auth, no local setup. Backlinks, keywords, domain ratings, batch analysis (100 URLs).

## What It Does

Backlink profiles, keyword research (volumes, difficulty, traffic potential), Domain Rating (0–100), batch analysis of up to 100 URLs, country-specific search volume data.

## Setup

Remote server with OAuth — no installation:
```json
{ "mcpServers": { "ahrefs": { "url": "https://mcp.ahrefs.com/mcp" } } }
```

Requires Ahrefs Lite ($129/mo+). 25,000 API units/month on Lite (min 50/call). 10-row limit on Lite plan.

## What's Good

Official Ahrefs product. Zero-friction remote OAuth setup. World-class SEO data (billions of pages indexed). Batch analysis makes it practical for agencies.

## What's Not

$129/mo minimum, no free tier. API units consumed on every call. 10-row cap on cheapest plan. Legacy local server deprecated.

## Rating: 4/5

One of the best-executed commercial MCP integrations. World-class data, zero-friction setup. Cost limits accessibility but justified for existing Ahrefs customers.
