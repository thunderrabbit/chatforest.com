**At a glance:** Analytics has the strongest vendor adoption of any MCP category. Google, PostHog, Amplitude, and Mixpanel all ship official servers.

## Enterprise Platforms

**[Google Analytics MCP](https://github.com/googleanalytics/google-analytics-mcp)** (1,500 stars, official) — 7 tools for reports, real-time data, Google Ads links. Read-only by design.

**[PostHog MCP](https://github.com/PostHog/mcp)** (143 stars) — 27+ tools in PostHog's monorepo. Deepest analytics MCP integration available.

**[Amplitude MCP](https://amplitude.com/mcp-server)** (official hosted) — 24+ tools with OAuth 2.0. Event analytics, cohorts, retention, experiments.

**[Mixpanel MCP](https://docs.mixpanel.com/docs/features/mcp)** (official) — OAuth-based hosted server.

## Privacy-First

**[Plausible MCP](https://github.com/AVIMBU/plausible-mcp-server)** (5 stars) — Minimal but functional. **[Matomo MCP](https://github.com/FGRibreau/mcp-matomo)** (8 stars, Rust) — Dynamic tool generation.

**Rating: 3.5/5** — Best vendor adoption across MCP. Privacy-first platforms lag.
