**At a glance:** 15+ servers, dedicated Genealogy-MCP GitHub organization. **Rating: 3.5/5.**

## GEDCOM File Handling

- **ancestry-mcp** (33 stars) — read, parse, search .ged files
- **GedcomMCP** (3 stars) — create, edit, query with relationship analysis

## Gramps Web (4 Implementations)

- **cabout-me/gramps-mcp** (25 stars, 16 tools) — smart search, data management, tree analysis. Self-hostable for privacy.
- **Genealogy-MCP/gramps-mcp** (19 tools) — fork with 3-category organization

## FamilySearch & WikiTree

- **FamilySearch MCP** (4 tools) — place search, date standardization, Family Tree search
- **WikiTree MCP** (10 tools) — profiles, trees, photos, categories. No auth required.

## Research & Analysis

- **research-sources-mcp** — Library of Congress + WikiTree + OpenArch.nl + Find A Grave in one search
- **tree-analyzer-mcp** — duplicate detection, chronological validation

## The Bottom Line

**Rating: 3.5/5** — Impressive community organization around Gramps + GEDCOM. Missing: commercial platform integrations, DNA analysis, OCR for historical documents.
