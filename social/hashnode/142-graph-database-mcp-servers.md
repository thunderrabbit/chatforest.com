**At a glance:** 15+ servers, 10+ graph databases. **Rating: 3.5/5.**

## Neo4j

- **[neo4j/mcp](https://github.com/neo4j/mcp)** (~188 stars, official) — Cypher access, multi-deployment
- **[neo4j-contrib/mcp-neo4j](https://github.com/neo4j-contrib/mcp-neo4j)** (~853 stars, Labs) — GraphRAG + knowledge graph memory

## Key Servers

- **Amazon Neptune** (AWS official) — openCypher + Gremlin
- **TigerGraph** — 34 tools, most comprehensive dedicated server
- **Dgraph** (~20,400 stars) — MCP built into core product (v25+)
- **Memgraph** — built-in PageRank, centrality, vector search
- **ArangoDB** — community only, up to 46 tools (async server)
- **FalkorDB** — Redis-based with GraphBLAS
- **ArcadeDB** — multi-model with built-in MCP

## Bottom Line

Most major graph databases have official servers. Dgraph and ArcadeDB ship MCP built-in. Missing: ArangoDB official server, JanusGraph/TerminusDB coverage. **Rating: 3.5/5.**
